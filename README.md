# Loan-data-analysis
This project focuses on analyzing loan data using NumPy to perform data cleaning, preprocessing, and statistical analysis. The objective is to extract meaningful insights related to loan approvals, interest rates, and potential risk patterns.

It demonstrates how numerical computing can be applied to real-world financial datasets for better decision-making.
Features
 Data loading and preprocessing using NumPy
 Handling missing values using statistical methods
 Descriptive statistical analysis (mean, median, standard deviation)
 Loan risk identification using conditional filtering
 Insight generation from financial data
 Exporting cleaned dataset for further use

 Tech Stack: NumPy & Pandas
